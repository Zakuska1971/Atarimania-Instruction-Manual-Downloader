# Atarimania-Instruction-Manual-Downloader

This is a tool I built entirely using COPILOT AI. It downloads the instruction Manuals from the Atarimania.com website for the Atari 2600 section, so I can build a user-friendly DVD with all the ROMs and Instruction Manuals included.  My brother and I already own many of these ROMs, and the rest are in the public domain.
